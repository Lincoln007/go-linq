go-linq

.NET's Language Integrated Query (LINQ) library for Go

Query and manipulate Go slices and collections using SQL-like methods.
Written in pure Go with no dependencies, tested with 100% code coverage.

Fork it! Wiki GoDoc

Bring your type

There are no generics in Go, but go-linq's magical hacks let you use your types with no trouble.

Method syntax

From(slice) .Where(wat) .Select(dat) .Results()
Easy as that. Just keep writing your queries.

Parallel LINQ

Your queries executed concurrently using goroutines under the hood by just calling .AsParallel().

Basic

Find average age of students over 20.

Import your []*Student. results, err := From(students)
Find those over 20. Note T is an alias for interface{}. So we make type assertion to our input type.     .Where(func (s T) (bool, error){
            return s.(*Student).age >= 20, nil
    })
Select their ages.     .Select(func (s T) (T, error){
        return s.(*Student).age, nil
    })
Find the average. .Average()

Concurrent

Check if all given websites have a favicon.ico.

  allHasFavicon, err := From(domains)
Convert domain names to full URLs.     .Select(func (in T) (T, error){
        return fmt.Sprintf("http://%s/favicon.ico", in), nil
    })
Let's run the following in parallel.     .AsParallel()
Find if all have the favicon.     .All(func (url T) (bool, error){
        resp, err := http.Get(url.(string))
        if err != nil {
            return false, err
        }
        return resp.Status/100 == 2, nil
    }
